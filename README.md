# Frontend — Los conflictos mundiales

Pequeña guía para arrancar el frontend de la práctica.

Requisitos
- Backend corriendo
- Base de datos arrancada

**Instalación y desarrollo**

1. Clonar el repo (git clone <url>).
2. cd frontend (Node.js v16+).
3. npm install
4. npm run dev

**Funciones**
- Crear un conflicto → aparece en la lista.  
- Editar desde la lista → Guardar → cambios persistidos.  
- Ver detalle y eliminar.
